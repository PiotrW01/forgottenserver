Detailed Description

This namespace provides functions to encrypt and decrypt data using the XTEA algorithm (See https://wikipedia.org/wiki/XTEA).

Decrypts data using the XTEA algorithm.

Parameters
{data}A pointer to the data to be decrypted. The decryption is performed directly on this buffer. The data must be in blocks of 8 bytes.
{length}The ength of the data to be decrypted, in bytes. It must be a multiple of 8.
{k}The expanded round keys used for decryption.
